Colors

This library defines 20 colours that are used by default. The default order is set by the l_colors list. A last colour : C_trensp is defined as a transparent colour and is not used in the colour list

#6307ba : Violet / Purple

#16b5fa : Cyan

#2ad500 : Vert clair / Light green

#145507 : vert foncé / Dark green

#ff8e00 : Orange

#cb0d17 : Rouge / Red

#5694b2 : Bleu pastel / Pastel blue

#569a57 #: Vert pastel / Pastel green

#b986b9 : Lavande

#c6403c #: Rouge pastel / Pastel red

#d39d5d : Beige

#25355d : Bleu / Blue

#fcc100 : Jaune / Yellow

#7ab5fa : Bleu ciel / Light blue

#fc2700 : Orange foncé / Dark orange

#0fc88f #: Bleu-Vert / Blue-Green

#a8173b : Rouge cerise / Red

#1812c4 : Bleu foncé / Dark blue

#000000 : Noir / Black

#707070 : Gris / Grey
